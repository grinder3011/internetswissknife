💡 How to use the Easy File Renamer:

  • 📂 Drag & drop files or click to select multiple files or folders.
  • 🔀 Choose how you want to sort the files (manual drag reorder, filename, date, or leading number).
  • ✏️ Enter a filename prefix, like vacation or invoice.
  • 🔢 Set the starting number for your files. Use leading zeros for padding (e.g., 0050).
  • ➕ Optionally add an underscore before the number for clarity.
  • 💾 Click Rename & Download ZIP to rename your files in batch and download them as a ZIP archive.

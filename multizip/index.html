📦 Quick-Multi Zip

💡 How to use this tool:

  • 📁 Use the file picker to select one or more files. You can add files incrementally — new files are added or replace existing ones with the same name.
  • 🗑️ Remove individual files from the list by clicking the ❌ button next to each filename.
  • 📦 Click Create Archive to prepare the zip file once you’re happy with your selection.
  • ⬇️ Use Download Archive to save the zip file to your device.
  • 🔄 Click Remove All Files to clear the entire selection and start fresh.
